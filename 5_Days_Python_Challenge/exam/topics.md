#  **Python Programming – Exam Topics**

**Day 1 – Fundamentals**
What is Python · Python uses & strengths · Installing Python / Colab · `print()` · Variables & assignment · Data types (`int`, `float`, `str`, `bool`) · Type conversion (`int()`, `float()`, `str()`, `bool()`) · `input()` · String formatting (f-string, `.format()`) · Comments (`#`, `'''`) · Built-ins `type()`, `len()`, `round()`, `abs()`

**Day 2 – Control Flow & Functions**

**Day 3 – Loops, Lists & Strings**
`for` loop · `while` loop · `break` / `continue` / `pass` · Iterating strings & lists · String indexing & slicing · Methods `.upper()`, `.lower()`, `.find()`, `.replace()`, `.split()`, `.join()` · Creating lists · Access & modify lists (`append`, `insert`, `remove`, `pop`) · List slicing · `in` keyword · `range()` function

**Day 4 – Data Structures & File I/O**
Tuples (immutable lists) · Sets (no duplicates, `add`, `remove`, `union`, `intersection`) · Dictionaries (keys & values, update, loop) · Nested dictionaries · File handling (`open`, `.read`, `.readlines`, `.write`, `with open`) · Basic modules (`math`, `random`, `datetime`)
